Data collection

We may record anonymous aggregated access logs when you interact with the site. For example, we may log the number of requests, type of requests, how the request was processed (cache hit, cache miss, 404, etc.), errors, referrers, and similar aggregated information. These logs do not contain IP addresses or other identifiable information.

When using projects that contain "cloud variables", your username (randomly generated or set by user), project ID, and some technical details may be logged. This information is deleted within 14 days. The data stored in cloud variables is purged within 24 hours of no users being connected to a given project. You can opt out of this data collection by disabling cloud variables.

By default, your username (randomly generated or set by user) is persisted between visits in your browser's "localStorage". To mitigate the possibility of uniquely identifying users with this feature, all randomly generated usernames are anonymized. However, usernames explicitly set by a user may be logged verbatim.

A self-hosted instance of Plausible Analytics is used to anonymously track site usage. You can opt out of this tracking by enabling your browser's "Do Not Track" setting. The data is publicly viewable at https://stats.turbowarp.org/turbowarp.org.

Data use

The data we collect is used to monitor usage, prevent abuse, optimize processes, etc.

Personal information that we collect is never shared or sold to any third party, unless necessary to provide the service (such as hosting providers) or to comply with local laws (like a government subpoena).
